# aoc-2022

Advent of Code 2022

# Every day of December

To solve a day :

- Add a solve.js file in days/[day]/ directory
- Add an input file in days/[day]/input.txt
- Export from solve.js part1, part2, part3...

# Usage

`yarn aoc [1-25]` run & solve the nth day
